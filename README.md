# ДонГУ Сеть

Социальная сеть Донецкого государственного университета.

## Возможности

- Авторизация через Email и OAuth 2.0 (Github, Google, Facebook)
- Редактирование профиля, фото профиля и обложки
- Создание, редактирование и удаление постов, комментариев и ответов
- Лайки постов, комментариев и ответов
- Прикрепление изображений и видео к постам
- Drag and drop сортировка медиа при создании поста
- Хэштеги в постах
- Упоминания пользователей (@mention)
- Бесконечная прокрутка ленты
- Подписки на пользователей
- Поиск пользователей с фильтрами
- Уведомления
- Галерея загруженных фото и видео
- Адаптивный дизайн
- Тёмная и светлая темы
- PWA (Progressive Web App) с офлайн поддержкой

## PWA (Progressive Web App)

Приложение поддерживает установку на устройства как нативное приложение.

### Возможности PWA

- Установка на домашний экран (Android/iOS/Desktop)
- Офлайн-доступ к кэшированным страницам
- Быстрая загрузка через Service Worker кэширование

### Генерация иконок

При изменении логотипа перегенерируйте PWA иконки:

```bash
npm run pwa:icons
```

### Установка приложения

**Chrome/Edge (Desktop/Android):**
- Откройте сайт в браузере
- Нажмите на иконку установки в адресной строке или в меню браузера "Установить приложение"

**Safari (iOS):**
- Откройте сайт в Safari
- Нажмите кнопку "Поделиться"
- Выберите "На экран Домой"

## Развёртывание

1. Настройте PostgreSQL и добавьте строку подключения в `.env`
2. Настройте OAuth провайдеры и добавьте ключи в `.env.local`
3. `npm install`
4. `npm run prisma:deploy`
5. `npm run build`
6. `npm run start`
